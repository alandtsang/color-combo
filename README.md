# Color Combo

服装颜色搭配工具，基于 60-30-10 色彩法则生成和谐配色方案。

## 功能特性

- **色轮选择器** - 通过点击色轮选择主色调
- **饱和度滑块** - 调整颜色饱和度
- **HEX 输入** - 直接输入色值
- **快速颜色** - 预设常用颜色快速选择
- **五种配色方案** - 互补色、类似色、三等分、拆分互补、双类似

## 技术实现

- 纯 HTML/CSS/JavaScript，无依赖
- HSL 色彩空间作为内部格式
- Canvas 绘制色轮，支持精确的颜色选择

## 配色算法

| 方案 | 色相偏移 |
|------|----------|
| 互补色 | +180° |
| 类似色 | ±30° |
| 三等分 | +120°, +240° |
| 拆分互补 | +150°, +210° |
| 双类似 | ±60° |

## 使用方式

直接在浏览器中打开 `index.html` 即可运行。

## 色彩转换工具

```javascript
hexToRgb(hex)     // #RRGGBB → {r, g, b}
rgbToHex(r, g, b) // → #RRGGBB
rgbToHsl(r, g, b) // → {h, s, l}
hslToRgb(h, s, l) // → {r, g, b}
hslToHex(h, s, l) // → #RRGGBB
```
